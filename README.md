🎓 Student Management System

📌 Project Overview
The Student Management System is a console-based Java application developed using Core Java and File Handling concepts. The project is designed to manage student records in an efficient and organized way through a simple menu-driven interface.

✨ Features
➕ Add Student
Users can add new student records by entering student ID, name, and age.

📋 Display Students
Displays all stored student records in a sorted and readable format.

🔍 Search Student
Allows searching of student details using the student ID.

❌ Delete Student
Enables deletion of student records from the system.

🚫 Duplicate ID Validation
Prevents insertion of students with the same ID.

💾 File Handling
Student records are permanently stored in a text file using Java File Handling.

🔄 Auto Data Loading
Previously saved records are automatically loaded when the program starts again.
